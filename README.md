# Team Expense Management System 💼

A **corporate expense management platform** built using Spring Boot and integrated with Stripe for secure reimbursement processing.

The system streamlines the **expense submission, approval, and payment workflow** inside organizations through role-based access for **Employees, Managers, and Admins**.

## 🚀 Workflow

### 1) Employee Portal

Employees can:

* Submit new expense requests
* Add expense details (amount, category, description)
* Upload receipts / proof documents
* Track submitted expense status
* View approval / rejection history

### 2) Manager Portal

Managers can:

* Review submitted expenses from team members
* Approve expenses
* Reject expenses with comments
* Monitor team expense requests
* Track pending approvals

### 3) Admin Portal

Admins can:

* View all approved expenses
* Manage reimbursement queue
* Process payments securely using Stripe
* Track payment status
* Generate organization expense reports
* Monitor company spending
